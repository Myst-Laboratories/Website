Mystara

Mystara is a competitive 2D platform fighter built from scratch in C++ using SDL. The game features fast-paced combat, rich character design, and immersive lore. It is the flagship project of Myst Laboratories and a showcase of our technical and creative abilities.

Lunari

Lunari is a personal budgeting and expense tracking application built with Java and Spring Boot. It enables users to manage their finances, set monthly spending goals, and visualize their income vs. expenses over time. Designed with simplicity and performance in mind, Lunari delivers an intuitive interface backed by robust backend services.
